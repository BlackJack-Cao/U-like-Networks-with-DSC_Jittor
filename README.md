# U-like-Networks-with-DSC_Jittor
This code implement the Jittor version of U-like Networks with DSC.

## Installation
1. install Jittor: `pip install jittor`

2. Oter requirement can be found at [here](https://github.com/BlackJack-Cao/U-like-Networks-with-DSC.git)
